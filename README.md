# Python Data Structures - Coursera

This repository contains my personal programming exercises and core assignment solutions for the **"Python Data Structures"** course, offered by the University of Michigan on Coursera.

## 📌 Repository Structure & Strategy
To keep the repository clean, compact, and highly readable, each chapter is organized into exactly **two distinct files**:
1. `basic_exercises.py` — A consolidated file bundling multiple fundamental exercises together to demonstrate core language mechanics.
2. `project_[name].py` — A dedicated, standalone file focused entirely on the single most interesting and complex algorithmic challenge of that chapter.

## 📂 Chapters Overview

### 🔹 Chapter-07: Files
* `basic_exercises.py`: Contains foundational tasks for reading text files, formatting text into upper case, and scanning logs to extract and count simple error codes.
* `project_spam_confidence.py`: A comprehensive script utilizing exception handling (`try/except`) to process external mail data, parse string boundaries securely, and calculate dynamic averages.

### 🔹 Chapter-08: Lists
* `basic_exercises.py`: Demonstrates sequential data management, capturing dynamic user inputs within a loop, and computing statistical averages using built-in list methods.
* `project_romeo_word_sorter.py`: A standalone algorithm that parses complex text files, systematically filters out all duplicate words, and sorts the final unique collection alphabetically.

### 🔹 Chapter-09 & Chapter-10: Dictionaries and Tuples
* `basic_exercises.py`: Bundles essential key-value frequency counters, showcasing word mapping and complex maximum loops to isolate top email senders within large network logs.
* `project_hourly_distribution.py`: My signature project for this course. It analyzes extensive communication logs, extracts timestamps, counts hourly frequencies via dictionaries, and uses the immutability of tuples to sort the output chronologically.
